# Claude Code 101

An introduction to Anthropic's agentic coding assistant, covering the fundamentals of using Claude Code to explore codebases, plan implementations, automate development workflows, and customize AI-assisted software engineering.

---

## Course Overview

Claude Code 101 introduces the core concepts behind Claude Code and teaches developers how to work effectively with an AI coding agent. The course covers agentic development workflows, context management, project customization, and integration with external tools to help developers build software more efficiently while remaining in control of the development process.

## Learning Objectives

After completing this course, you should be able to:

- Explain how Claude Code differs from traditional AI assistants.
- Understand the fundamentals of agentic software development.
- Use the Explore → Plan → Code → Commit workflow.
- Manage Claude Code's context effectively.
- Customize Claude Code using `CLAUDE.md`, Subagents, Skills, MCP, and Hooks.
- Integrate Claude Code into modern software development workflows.
- Improve development productivity while maintaining developer oversight.
