# Claude 101 – Module 4: Putting It All Together

## Summary

This module shifts from learning Claude's features to understanding **how to apply them in real-world work**. Rather than focusing on prompts or technical capabilities, it demonstrates practical workflows organized by profession and introduces the growing ecosystem of Claude products that integrate directly into the tools people already use.

The key idea is simple:

> Claude isn't just a chatbot—it's an intelligence that can work wherever your work already happens.

---

By the end of this module you should be able to:

- Identify practical Claude use cases you can start using immediately.

# Part 2 – Claude Beyond Claude.ai

One of the biggest ideas in this module is:

**Claude.ai is only one interface.**

Claude is becoming available everywhere people already work.

Each product is optimized for a different workflow.

---

# Claude.ai

Best for:

- Writing
- Research
- Analysis
- Brainstorming
- General productivity

Runs on:

- Web
- Desktop
- Mobile

---

# Claude Code

Purpose:

AI software engineering partner.

Runs inside:

- Terminal
- IDE
- Browser
- Slack

Use it when you want to:

- Build features from natural language
- Debug code
- Explore unfamiliar codebases
- Run tests
- Create commits
- Automate developer tasks

Think of it as an AI teammate that understands your codebase.

---

# @Claude (Slack)

Purpose:

Bring Claude directly into Slack conversations.

Use it to:

- Summarize threads
- Draft replies
- Prepare for meetings
- Research during conversations
- Launch Claude Code sessions from discussions

Ideal for team collaboration.

# Biggest Lesson

Claude is evolving beyond a single chatbot into an ecosystem of specialized tools.

Instead of asking:

> "Can Claude do this?"

The better question becomes:

> **"Which Claude interface is best for this task?"**

Using the right interface allows Claude to work with the context, files, and workflows already present in that environment, making it significantly more effective than copying and pasting everything into a chat.
